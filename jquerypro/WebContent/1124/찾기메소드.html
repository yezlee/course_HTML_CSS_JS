부모/조상찾기

요소.parent() : 해당 요소의 부모찾기
요소.parents() : 해당 요소의 조상들찾기
요소.parents(selector) : 해당 요소의 selector로 지정된 조상찾기
요소.parentsUntil(selector) : 해당 요소와 selector사이의 조상찾기
div (great-grandparent)
    ul (grandparent)
  • li (direct parent) span

후손찾기

요소.find(selector) 해당요소의 후손중 selector를 찾는다
$('box2').find('span')
div (great-grandparent)
    ul (grandparent)
  • li (direct parent) span

부모 비교 하기

요소의 부모.is(selector) p태그의 부모가 div인지 비교해서 맞다면 p태그를 복사하고 글자색을 변경하시오.
$('.box3 p').on('click', function(){
     if($(this).parent().is('.pd')){
         $(this).clone().appendTo( $(this).parent() )
     }
})
