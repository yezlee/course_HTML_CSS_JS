charAt(index) : 인덱스 위치의 문자를 얻는다.
==> index 위치의 문자를 구한다. index가 문자열의 범위를 벗어나면 빈 문자열이 리턴된다.

charCodeAt(index) : 인덱스 위치의 유니코드(아스키코드값) 값을 얻는다.
==> index 위치의 문자에 대한 유니코드를 구한다.

indexOf(item) : item(문자/문자열)의 위치를 얻는다.
==> indexOf(searchvalue, start)
==> 부분 문자열의 위치를 검색한다. start는 검색 시작 위치이며 생략시 0이 적용되어 처음부터 검색한다. 없을 경우 -1을 리턴한다.

입력한 값이 공백인지 아닌지 체크한다.
공백이면 다시 입력하도록 한다.

substring(from, to) : start ~ end-1 까지의 문자열을 추출
==> 두 위치 사이의 부분 문자열을 추출한다. to를 생략하면 뒤쪽 모든 문자열을 추출한다.

slice(start, end) : start ~ end-1 까지의 문자열을 추출
==>start 위치에서 end 위치까지 부분 문자열을 추출한다. 음수로 끝에서부터 위치를 지정할 수 있다.

substr(start, length) : start에서 length만큼의 문자를 추출
==> start에서 시작하여 length 길이만큼 부분 문자열을 추출한다. 길이를 생략하면 뒤쪽 모든 문자열을 추출한다.

String

split(separator, limit) : 구분자를 기준으로 문자열을 분리하여 배열로 리턴한다.
limit 갯수는 생략 가능한데, 생략하게 되면 문자열의 끝까지 수행하게 된다. 갯수를 정하면 갯수만큼만 수행한다.
==> 구분자로 구분된 문자열을 분리하여 배열로 리턴한다. limit는 최대 몇 개까지 리턴할 것인가를 지정한다.
