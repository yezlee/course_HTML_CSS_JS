box-sizing : 브라우저의 크기가 줄어들 때, 브라우저의 크기에 맞게 각 요소의 크기를 조절할 때 필요하다.
 
 요소의 가로 크기 width : 50% -> calc() => calculator로 계산한다!
 calc(50% - 0.5%) : - (minus)할때 반드시 공백이 있어야한다.
 
 width : calc(50% - 0.5%)
 width : calc(300px - 1%)
 width : calc(100px - 1px)
 

 전체길이 300px.
 두개의 div가 각각 50%를 차지한다.

 - box-sizing : border-box 일때 border와 padding 속성을 포함해서 크기를 50%로 한다.

box-sizing : border-box

    내용을 기준으로 해서 크기를 50% 차지하기 때문에 실제로 차지하는 건 50%가 넘지 않는다.
