input type ="button" 요소는 자체 기능이 없다.
 기능을 부여하기 위해서 script를 이용한다.
 기능은 직접 input태그안에 줘도 되고(인라인스크립트) style처럼 위에서 <script> 이렇게 해서 줄수도 있음	
 
 script에서 form의 요소에 접근하기
 - form의 name속성을 이용하기
 => document.ff.price
 => document.ff.qty
 
 - form요소의 값을 가져오기 (value)
 => document.ff.price.value
 => document.ff.qty.value
물품가격 :
수량 :

<input type ="button" value = "확인 (무조건만원)" onclick="alert('10,000원 입니다.')">


<input type ="button" value = "확인 (calc)" onclick="calc()">
