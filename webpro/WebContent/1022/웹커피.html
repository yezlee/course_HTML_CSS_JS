블럭요소 : 화면 가로 길이를 끝까지 차지하는 요소
					자동으로 줄이 바뀐다.
					h1 ~ h6, p, div, hr, pre...
			
			인라인요소 : 지정한 범위만큼만 차지한다. 자동으로 줄이 바뀌지 않는다.
					 sup, sub, span, br, i, em, strong, img
						
			margin : 테두리 바깥쪽 여백
			padding : 테두리 안쪽 여백 
			
			블럭요소를 옆으로 배치하기 위해서 가로길이를 조절한다. 
			float : left OR right 스타일을 지정한다.
			
			div{
				border : 1px solid orange;
				padding-left : 50px; /*패딩은 안쪽으로 마진은 바깥쪽으로 여백*/
				margin : 50px; 
				width : 40%;
				float : left;
			}
