요소찾기

item의 위치값을 넣는다. start위치부터 item의 위치값을 찾는다.
indexOf(item), indexOf(item, start) - 얘는 앞에서 부터
lastIndexOf(item), lastIndexOf(item, start) = 얘는 뒤에서부터 아이템을 찾는다

추가/삭제

push(item) : 배열의 마지막 요소에 item을 추가한다.
pop() : 배열의 마지막 요소를 제거한다.

sort

nums = [234, 10, 134, 250, 300, 3]
nums.sort() : 자릿수별 숫자크기를 판별 -> 10, 134, 234, 250, 3, 300
오름차순 : nums.sort(function(a,b){ return a-b; });
내림차순 : nums.sort(function(a,b){ return b-a; });

splice - 수정(삭제/추가)

삭제해서 수정이 되기도하고, 추가해서 수정이 되기도 함
splice(index , n) - 몇번째 index위치에서 n개를 삭제하라.
splice(index, n, a, b, c) - 몇번째 index위치에서 n개를 삭제하고 a, b, c를 추가하라
splice(index, 0, a, b, c) - 몇번째 index위치에서 삭제하지 않고 a, b, c를 추가하라
